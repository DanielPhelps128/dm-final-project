Project Summary (Updated)

This project analyzes the Ames Housing dataset using techniques from class to provide an interpretable, reproducible study. The final scope includes: (1) EDA and clear visualizations, (2) PCA to summarize structure, and (3) k-means clustering to form market segments. We intentionally stopped before advanced prediction so the results stay transparent and aligned with the course topics.

Scope & Methods

  • EDA & Visualization: price distribution, scatterplots, box plots.
  • PCA: scree plot, PC1–PC2 mapping, loading interpretation.
  • Clustering: k-means on PCA scores; silhouette-based K selection; segment profiles.

Key Results (Brief)

  • PC1 = size/quality gradient.
  • PC2 = layout/vertical structure.
  • Clusters: entry, mid-market, premium segments with clear price and quality differences.
