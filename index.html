Overview & Motivation

Goal: explain what drives higher vs. lower home prices using only topics from class. The pipeline is simple—clean the data, explore with visuals, reduce dimensions (PCA), find natural groups (clustering), and mine combinations of features tied to price bands (association rules). If an approved listing dataset includes short descriptions, I will add basic text features (TF‑IDF + sentiment).

Dataset

Primary structured data: Ames Housing (Kaggle).

  • Target: SalePrice (also summarized in Low / Medium / High bands for rules)
  • Features: living area, lot area, quality/condition scores, year built/remodeled, neighborhood, etc.
  • Usage: follow Kaggle license/terms

Methods (course topics)

  • EDA & Visualization: distributions, box plots by neighborhood/quality, correlation heatmaps
  • Dimensionality Reduction: PCA and a t‑SNE 2‑D map
  • Clustering: k‑means & hierarchical; choose k via silhouette; profile clusters
  • Association Rules: Apriori / FP‑Growth on categorical + discretized features; support, confidence, lift
  • Anomaly checks: z‑scores or distance from clusters; compare with/without outliers
  • Optional Text: TF‑IDF keywords + simple sentiment (VADER/AFINN) if descriptions are approved

Planned Experiments

  1. E1: Clean data; create price bands; EDA with short captions
  2. E2: PCA + t‑SNE visualizations
  3. E3: Clustering (k‑means & hierarchical) and cluster profiles
  4. E4: Association rules for High/Low price bands (support, confidence, lift)
  5. E5: (optional) Add TF‑IDF + sentiment; re‑check clusters/rules
  6. E6: Outlier/sensitivity analysis (with vs. without flagged outliers)
