Predicting Housing Prices by Combining Structured Data and Text

View Proposal GitHub Repo Dataset

Overview & Motivation

This project evaluates whether listing descriptions (text) improve housing price prediction when combined with structured features (e.g., rooms, sqft, location). I compare linear models, tree ensembles, and simple neural networks, measuring performance with RMSE/MAE/R².

  • Objective: quantify the added value of text features
  • Scope: data cleaning, feature engineering, modeling, and interpretability
  • Deliverables: code, 3-page proposal, final report, slides

Dataset

Primary dataset: Kaggle Housing Prices (structured attributes) plus listing descriptions if available, or a comparable dataset that includes text.

Methods

  • Preprocessing: missing-value handling, scaling, train/validation/test split
  • Text: tokenization, stop-word removal, TF-IDF and/or embeddings
  • Models: Linear/Ridge, Random Forest, XGBoost, simple MLP
  • Evaluation: RMSE, MAE, R²; k-fold CV
  • Interpretability: feature importance, SHAP (optional)

Planned Experiments
